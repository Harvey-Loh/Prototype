Going concern signals

Scans the MD&A Markdown for hedging language ("may", "material uncertainty", "ability to continue") and cross-checks against liquidity and solvency ratios to surface risk before fieldwork starts.

NLP + ratio thresholds
Module 04

Evidence auto-vouching

Client uploads supporting document PDFs (invoices, bank statements, contracts). The pipeline extracts key fields — date, counterparty, amount, reference — into a structured table ready to paste directly into the auditor's working paper. Eliminates manual transcription from support documents.

PDF extraction · copy-paste ready

Single number · pandas

Risk
Risk & fraud signal detection
Industry-standard forensic models run automatically on every client
Beneish M-Score Altman Z-Score YoY variance flags
Module 01

Beneish M-Score

8-variable model that flags earnings manipulation risk. Score above −1.78 is a warning sign. Public formula, fully computable from Step 02 output — no additional data needed.

Public formula · ~1 day to build
Module 02

Altman Z-Score

Predicts financial distress probability using five weighted financial ratios. Score below 1.81 signals significant bankruptcy risk. Computable directly from the extracted balance sheet and P&L.

Public formula · pandas
Module 03

YoY variance flags

Automatically highlights any line item that moved more than 20% year-on-year. Flagged in a dedicated Excel sheet with the exact amount and percentage change — no manual scanning.

Pure pandas · no AI
Module 04

MD&A sentiment scan

Compares the MD&A Markdown between two years. Detects increases in hedging language, new risk factors, and changes in tone that may signal management concern before it shows in numbers.

Claude API · Markdown input
Module 05

Segment analysis

Extracts segment revenue, profit, and asset breakdowns where disclosed. Tracks revenue mix shifts over time — a leading indicator of strategic risk or concentration.

Step 02 segment tables
Module 06

Covenant monitor

Extracts debt covenant terms from the notes (minimum ratios, restriction thresholds) and computes current headroom against each covenant — flags approaching breaches early.

Notes parsing · ratio calc

Checks sustainability report Markdown against IFRS S1, IFRS S2, and TCFD framework requirements. Returns a compliance gap report per client — ready to share or use as a workpaper.

Consistent with Phase 1 format
