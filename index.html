Annual Report
PDF Extractor

Converts annual report PDFs into structured Excel audit workpapers by extracting Markdown text and tables using AI-powered document parsing.

picture_as_pdf
Step 01
PDF → Markdown

Split into 50-page chunks and processed by the docling AI engine, extracting text and table structure.

table_chart
Step 02
Markdown → Tables

Cross-page tables stitched together. Financial statement items matched to known abbreviations automatically.

grid_on
Step 03
Tables → Excel

Each table becomes a named Excel sheet. Brackets converted to negatives, index sheet auto-generated.

What it extracts
check_circle Financial statement tables (5 core statements)
check_circle Notes to financial statements
check_circle Structured Markdown text content
check_circle Financial numbers (brackets → negatives)
Current limitations
cancel Charts, graphs and embedded images
cancel Handwritten or stamp annotations
cancel Fully scanned PDFs (lower accuracy via OCR)
cancel Multi-column free-text narrative sections
warning
Prototype Scope — Please Read Before Use
Extraction quality depends on PDF structure. Digital PDFs are recommended. Scanned PDFs are supported via OCR but may produce lower quality results. All extracted data should be reviewed and verified before use in audit work.
